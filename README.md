# RJ Psicoterapia Online 🧠

Landing page moderna e responsiva desenvolvida para o psicólogo clínico Rogério Jefferson (CRP 06/104438), focada em captar pacientes para atendimento online (Psicoterapia e Psicanálise).

## 🎯 Objetivo do Projeto
Proporcionar uma presença digital profissional, transmitir autoridade clínica e facilitar o agendamento de consultas online diretamente via WhatsApp, utilizando técnicas de copywriting focadas em conversão.

## 🚀 Tecnologias Utilizadas
* **HTML5:** Estruturação semântica e SEO-friendly.
* **CSS3 (Vanilla):** Estilização customizada, variáveis de cor, layout responsivo (Flexbox/Grid) e animações CSS.
* **JavaScript:** Interatividade da página, animações de scroll (Intersection Observer) e rolagem suave (Smooth Scroll).

## ✨ Funcionalidades
* **Design Responsivo:** Adaptável para Desktop, Tablets e Smartphones.
* **Scroll Animado:** Elementos aparecem suavemente conforme o usuário rola a página.
* **Botões Flutuantes:** CTAs estratégicos para o WhatsApp com mensagens automáticas pré-configuradas.
* **SEO Básico:** Meta tags configuradas para ranqueamento em motores de busca.
